Investment Villas — виджет для сайта

Готовый блок «инвестиционный каталог вилл» для ava.estate: карточки в списке + развёрнутая страница с показателями доходности и графиками. Данные подключаются одним файлом и обновляются автоматически.

Инструкция по интеграции · для веб-разработчика

01Что это

Самодостаточный виджет (HTML/CSS/JS, без внешних зависимостей): список карточек вилл → тап по вилле открывает детальную страницу с описанием, ценой и блоком «Investment performance» (доходность/год, чистыми в месяц, занятость, помесячный график, накопленная доходность). Дизайн уже в стиле ava.estate (тёмная плитка + кремовый + Manrope).

Живое демо

Кликабельно, с реальными данными.

avaestate.github.io/ava-rental/b2b/

Исходники (index.html + data.json): репозиторий avaestate/ava-rental, папка /b2b/.

02Быстрая вставка (Tilda) — 5 минут

Самый быстрый путь — встроить готовый виджет блоком «HTML» (T123 / Zero-block «Embed»). Вставьте этот код:

<!-- AVA Investment Villas -->
<div style="max-width:480px;margin:0 auto">
  <iframe src="https://avaestate.github.io/ava-rental/b2b/"
    style="width:100%;height:1400px;border:0;border-radius:24px"
    loading="lazy" title="AVA Investment Villas"></iframe>
</div>
Высоту height подстройте под число вилл (≈150px на карточку). Виджет адаптивный — на телефоне займёт всю ширину. Внутренняя навигация (список ↔ детальная) работает прямо внутри блока.

03Данные — один редактируемый источник

Виджет читает единственный файл data.json (лежит рядом: /b2b/data.json). Чтобы изменить цифры — правится только он, вёрстку трогать не нужно.

A
Автообновление из Guesty. Раз в день скрипт на нашей стороне пересчитывает реальные показатели по бронированиям и обновляет data.json сам. Разработчику здесь делать ничего не нужно.
B
Ручные правки / витрина. Значения в data.json можно переопределять вручную (какие виллы показывать, цена продажи, целевая доходность). Для нетехнической правки заведём Google-таблицу → она будет генерировать data.json.

04Глубокая интеграция (по желанию)

Если хотите не iframe, а нативные карточки Tilda в общем каталоге — возьмите готовый компонент (разметка карточки и детальной + CSS) из /b2b/index.html как эталон и рендерите те же поля из data.json в свою систему карточек. Логика графиков — чистый inline-SVG, без библиотек.

05Что важно

  • Виджет полностью автономен: нет внешних скриптов/CDN, только шрифт Manrope (Google Fonts).
  • Адаптивный, страница по горизонтали не скроллится.
  • Данные и вёрстка разделены — контент меняется без правки кода.
  • Фото сейчас тянутся из нашей системы (Guesty CDN) — стабильные ссылки.
